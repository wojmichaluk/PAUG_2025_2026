# PAUG_2025_2026
Repository for PAUG (Praktyczne aspekty uczenia głębokiego - Practical aspects of deep learning) course at AGH University

Repozytorium zawiera rozwiązania zadań z kolejnych laboratoriów (jupyter notebooki **.ipynb**) wraz z poleceniami (pliki **.pdf**) - katalogi **lab*i***, `i = 1, 2,..., 6`.
Do laboratorium nr 2 jest dołączony także katalog *dataset_cat_dogs* odpowiadający używanemu datasetowi, zawierający odpowiednio podzielone na podkatalogi pliki ze zdjęciami.
Jest on wymagany także w laboratorium nr 3, tylko tam pod nazwą *cats_and_dogs_small* - nie dodawałem go drugi raz.
Ponadto w laboratorium nr 6 są wymagane pliki, do których URL jest podany w komentarzu w pierwszej komórce notebooka (są one także wspomniane w instrukcji).
Reszta używanych zbiorów do treningu powinna być dostępna z poziomu importów używanych bibliotek.
Poza tym, w laboratorium nr 6 uwzględniłem plik **script.py**, na którym jest oparte rozwiązanie (notebook).

Oczywiście rozwiązania zadań mogą zawierać błędy czy nie obejmować wszystkich punktów polecenia, ale mam nadzieję, że okażą się przydatne. Korzystajcie rozważnie!
